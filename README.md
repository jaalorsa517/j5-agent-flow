# j5-agent-flow

MCP Server for Topologic Agent Pipeline orchestration.

## Description
This MCP Server is designed to orchestrate agent workflows following a strict topological pipeline.

## Install
1. Clone the repository
2. Run `pnpm install`
3. Run `pnpm build`

## Usage
Configure the MCP server in your client to point to the built `j5-agent-flow/index.js` file.
